# BI RAG PDF Assistant

A **Business Intelligence RAG-based PDF Assistant** that allows users to upload PDFs and ask questions. Built with Python, Streamlit, OpenAI, LangChain, and ChromaDB.

## Features
- Upload PDF files and extract text
- Split large documents into chunks
- Store embeddings using ChromaDB vector store
- Ask questions using OpenAI LLM
- RAG (Retrieval-Augmented Generation) based answers

## Tech Stack
- Python 3.11
- Streamlit
- OpenAI API
- LangChain
- ChromaDB
- PyPDF2
